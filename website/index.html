Pollo, the XML editor

What is it?

Pollo lets you edit XML files. Right now it's best suited for files which can be represented in a tree structure. It comes with out-of-the-download support for Cocoon sitemap files, Ant build files and XML Schemas.

The editor has a colourfull tree widget (not the default Swing one!), drag-and-drop support, unlimmited undo, model-view design (open multiple views on the same file), ...

Screenshots: editing sitemap file , editing an xml schema.

What's the status?

Usable but experimental, and lacking some very usefull features (keyboard support, plugins, recent files menu, ...).

Platforms?

Pollo is written for Java 1.3. I mostly developed it on Linux but have done some testing on Windows also. Make sure you have a recent JDK version (but not 1.4).

How do I get it?

Download the zip at the sourceforge project site. If you have problems extracting it, use jar: jar xvf pollo-0.2.zip
