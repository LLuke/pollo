Pollo: MVC

Pollo's internal design is based on a pattern called 'MVC' (Model-View-Controller). From a user perspective, this allows you to open multiple views on the same file.

Pollo follows the trend seen in modern webbrowsers such as Mozilla by allowing views to be created either in new windows, or as new tab within the same window.

By default, when opening a file, it is added as a new tab to the current window.

To open a file in a new window, first create a new window by choosing View | New Frame in the menubar, and then open the file by choosing File | Open from the menubar in the newly created window.

To create a new view on an already opened file, choose View | Create new view on file | (file name).
