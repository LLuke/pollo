Pollo: display specifications

A "display specification" is the thing that defines what colors are used in Pollo's tree view. Additionally, it can specify the ordening of the attributes. This is because in the data model used by Pollo (the DOM), the attributes have no order. However, it is usually convenient to see attributes in some specific order.

There are multiple display specification implementations available:

  • One that reads its behaviour from a file
  • One that uses a fixed color
  • One that randomly assigns colors to elements

Additionally, it is possible to write your own.
